# Nykaa Marketing Campaign Data Analysis

## Project Overview

This project analyzes Nykaa's marketing campaign performance using Python and Data Science techniques. The objective is to evaluate campaign effectiveness, customer engagement, return on investment (ROI), and revenue generation across different marketing channels and target audiences.

The analysis helps identify the most successful campaign types and marketing channels, enabling data-driven business decisions.

---

## Dataset Information

The dataset contains marketing campaign details such as:

- Campaign ID
- Campaign Type
- Target Audience
- Duration
- Channel Used
- Impressions
- Clicks
- Leads
- Conversions
- Revenue
- Acquisition Cost
- ROI
- Language
- Engagement Score
- Customer Segment
- Date

Total Records: 55,555

## Project Workflow

### 1. Data Loading
- Imported dataset using Pandas.
- Checked dataset structure and dimensions.

### 2. Data Cleaning
- Checked missing values.
- Removed duplicate records.
- Verified data types.

### 3. Feature Engineering
Created additional performance metrics:

#### Click Through Rate (CTR)
CTR = (Clicks / Impressions) × 100

#### Conversion Rate
Conversion Rate = (Conversions / Clicks) × 100

#### Cost Per Click (CPC)
CPC = Acquisition Cost / Clicks

#### Revenue Per Conversion
Revenue Per Conversion = Revenue / Conversions

### 4. Exploratory Data Analysis (EDA)

Performed statistical analysis using:
- Dataset summary
- Descriptive statistics
- Correlation analysis

### 5. Data Visualization

Generated visualizations including:

#### ROI Distribution by Campaign Type

- SEO
- Social Media

#### Top Marketing Channels by Revenue
Bar chart displaying average revenue generated by top-performing channels.

#### Audience Conversion Analysis

#### Correlation Heatmap
Heatmap showing relationships between numerical variables such as:
- Impressions
- Clicks

- Conversions
- Revenue
- ROI
- Engagement Score

## Key Insights

- Social Media and Influencer campaigns generate strong customer engagement.
- Certain marketing channels contribute significantly higher revenue.
- Higher clicks generally lead to increased conversions.
- ROI varies across campaign types and marketing strategies.
- Customer targeting plays an important role in campaign success.
